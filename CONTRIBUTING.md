# Contribuindo para VocalCore

Obrigado por querer contribuir! Abaixo estão passos rápidos para configurar um
ambiente de desenvolvimento e submeter alterações.

Pré-requisitos
- Swift 5.10+

Executando os testes

```bash
swift test
```

Estilo de código
- Siga as convenções Swift API Design Guidelines.
- Procure manter funções pequenas e testáveis.

Fluxo de trabalho para PRs
1. Abra uma branch a partir de `main` com nome claro (`feature/`, `fix/`).
2. Rode `swift test` e certifique-se de que novos testes passam.
3. Abra um PR descrevendo a motivação e o que foi alterado.

Issue & Bug reports
- Crie uma issue com passos para reproduzir e, se possível, artefatos de áudio
  ou dados sintéticos que exemplifiquem o problema.

Contato
- Abra uma issue para dúvidas rápidas ou discuta mudanças maiores antes de começar.
